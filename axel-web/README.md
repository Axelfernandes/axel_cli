# Axel Web

Browser-based AI coding assistant that works directly with GitHub repositories.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
```bash
cp .env.local.example .env.local
# Edit .env.local with your API URL and auth secrets
```

3. Run development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## Features

- Browse GitHub repositories
- Chat with AI about your code
- View and edit files
- Create branches and pull requests
- Works entirely in the browser - no local installation required
